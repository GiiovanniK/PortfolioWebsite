By Chasseur

Website- development & design

For one of my clients, By Chasseur, we were asked to design and develop a completely new company website. The website needed to be editable by the client themselves, so they wouldn't have to ask us to make changes in the code. Using the back-end CMS (Content Management System) WordPress, they can modify text and images. In addition, the website required an e-commerce component, which we implemented using Shopify.

Figure 1: Home

For this project, I primarily focused on programming the website using HTML, CSS, JavaScript, and PHP. I also collaborated with the designers to refine some designs, making them more logical and easier to program. Once the website was fully programmed, we integrated it with WordPress to ensure that parts of the website would be editable by the client. Afterwards, we developed the e-commerce component using Shopify, customizing it to align with the look and feel of the WordPress website.

Figure 2: Shop

In both the WordPress and Shopify back-end, the client can customize various sections of the websites, including texts, images, links, and the number of activity cards displayed. Additionally, the client has full control over customizing the main menu and footer of the websites.

Figure 3: Back-end CMS

Naturally, the websites were fully designed and developed with responsive mobile functionality in mind. It scales seamlessly across all devices to provide the best possible user experience.
